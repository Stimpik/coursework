# Резервное копирование фотографий VK → Яндекс.Диск

Консольное приложение для резервного копирования фотографий профиля пользователя из VK в облачное хранилище Яндекс.Диск.
Программа получает изображения через API социальной сети, выбирает фотографии максимального размера и загружает их в отдельную папку облачного хранилища.
Дополнительно формируется JSON-файл с информацией о сохранённых изображениях.
